
# Ticket Bounty

A full-stack ticket management application built with **Next.js 15** and **React 19**, focusing on performance, type safety, and modern server-side features.


## Key Features

  * **Modern Architecture:** Leverages React Server Components (RSC), Server Actions, and Partial Pre-Rendering.
  * **Full-Stack Capabilities:** Integrated API routes and efficient CRUD operations.
  * **Secure Authentication:** Custom auth implementation using **Oslo** (JWT/OAuth) with Role-Based Access Control.
  * **Robust Validation:** End-to-end type safety and data integrity using **Zod**.
  * **Optimized UX:** Features caching strategies, streaming, lazy loading, and a responsive UI (Light/Dark mode).
